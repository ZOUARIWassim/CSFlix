# 🎬 CSFlix

> Movie Recommendation Web Application

CSFlix is a full-stack web application that allows users to discover movies and receive personalized recommendations based on their preferences.

The project combines **frontend, backend, and API layers** to deliver an interactive movie browsing experience. :contentReference[oaicite:0]{index=0}

---

## 🚀 Features

- 🎥 **Movie Discovery**
  - Browse and explore movies
- 🤖 **Recommendation System**
  - Suggest movies based on user preferences
- 🌐 **Full-Stack Application**
  - Integrated frontend + backend + API
- ⚡ **Interactive UI**
  - Smooth and responsive user experience
- 📊 **Data-Driven**
  - Uses datasets and logic for recommendations

## 🧠 How It Works

CSFlix provides recommendations by combining:

- User interactions (search / selection)
- Movie metadata
- Recommendation logic (content-based or collaborative filtering)

This enables a **personalized movie discovery experience**.

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ZOUARIWassim/CSFlix.git
cd CSFlix
```

---

### 2. Run Backend

```bash
cd Backend
pip install -r requirements.txt
python app.py
```

---

### 3. Run Frontend

```bash
cd Frontend
npm install
npm start
```

---

### 4. API (if separate)

```bash
cd API
# Run API service depending on implementation
```
